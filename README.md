#  Dalavai Hemanth 

A premium, interactive 3D developer portfolio showcasing software engineering, data science, and machine learning projects. Built using React, Vite, Three.js, and Framer Motion, it features a glassmorphic user interface, smooth kinetic scrolling, dynamic micro-animations, and ambient audio feedback.

🔗 **Live Deployment:** [dalavaihemanth.vercel.app](https://dalavaihemanth.vercel.app)

---

## ✨ Features & Design Aesthetics

- **Immersive 3D Backgrounds:** Built with Three.js (`@react-three/fiber` and `@react-three/drei`) for dynamic, interactive background depth.
- **Micro-Animations & Transitions:** Fluid transitions, interactive card layouts, and layout animations powered by Framer Motion.
- **Glassmorphic UI Design:** High-contrast design styled with curated HSL gradients, frosted glass panels, and ambient glowing highlights.
- **Kinetic Scrolling:** Integrated Studio Freight Lenis for smooth scrolling, enhancing overall visual control and page pacing.
- **Tactile Audio Feedback:** Seamless UI sound effects (synthesized frequency ticks on button/card hovers and clicks).
- **Custom Hardware Cursor:** Dynamic interactive cursor modes that scale and change shape depending on interactive page elements.

## 💻 Local Installation & Setup

Ensure you have [Node.js](https://nodejs.org/) installed, then follow these commands:

```bash
# 1. Clone this repository
git clone https://github.com/DalavaiHemanth/Portfolio.git

# 2. Navigate to the project folder
cd Portfolio/portfolio-3d

# 3. Install dependencies (using legacy peer deps for React 19 / 3D library compliance)
npm install --legacy-peer-deps

# 4. Start the local development server
npm run dev
```

The application will launch locally at `http://localhost:5173/`.
